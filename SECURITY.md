# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ai-toolbox, please report it privately
by opening a GitHub Security Advisory:

https://github.com/wolkat/ai-toolbox/security/advisories/new

Please do **not** report security vulnerabilities via public GitHub issues.

You should receive an acknowledgement within 48 hours.

## Scope

This policy covers the configuration files, scripts, and templates in this
repository. Secrets or credentials accidentally included in commits should
be reported immediately.
